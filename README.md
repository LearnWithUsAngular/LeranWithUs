
# Learn With Us

App Like Udemy with MEAN Stack.



![Logo](https://miro.medium.com/max/1400/1*xtBtUvSRLn6KmM6mWeF82Q.png)


## Run Locally

Clone the project

```bash
  git clone https://github.com/LearnWithUsAngular/LeranWithUs.git
```

Backend

```bash
  cd backend
  npm install
  npm run watch
  npm run dev
```

Frontend

```bash
  cd Frontend
  npm install
  ng serve
```
